边框

预定义了几组间距样式。

默认边框

默认边框.b定义了1px的边框，并默认赋予了灰色。.b-t、.b-r、.b-b、.b-l分别定义了四条单边的边框

.b
.b-t
.b-r
.b-b
.b-l

边框大小

除了默认的1px边框，还分别定义了2-5px的边框

.b-2x
.b-3x
.b-4x
.b-5x

边框颜色

除了默认的1px边框，还分别定义了2-5px的边框

.b-light
.b-dark
.b-primary
.b-success
.b-info
.b-warning
.b-danger
.b-black
.b-white

虚线边框

搭配.b-dashed可以得到虚线边框

.b .b-5x .b-primary .b-dashed

圆角边框

.r-sm
.r-md
.r-lg
.r-round
.r-circle
