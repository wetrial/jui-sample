高宽

系统预定义了一些宽度跟高度样式类。包括固定大小和百分比大小。

预定义固定高度

h-xs 30px，可用于顶部较矮的导航栏topbar
h-sm 50px，一般用于后台页面的顶部工具栏toolbar
h-md 60px，一般用于前台页面的导航栏
h-lg 80px，一般用于前台页面的导航栏
h-xl 150px，可用于顶部或底部的滑出栏

预定义固定宽度

w-xs 30px，可用于侧边快捷指示栏
w-xs 60px，可用于侧边单级图标菜单栏
w-md 150px，可用于侧边树形菜单栏
w-lg 200px，可用于侧边树形菜单栏
w-xl 300px，可用于侧边树形菜单栏
