jlist 插件用于集合数据的显示。集合数据可以是普通数组，也可以是分组数据，还可以是层级（树形）数据。

Options

名称 类型 默认值 描述
url string null 获取远程数据的url地址。返回的数据必须是一个对象数组或者是一个包含Items属性对象。
dataFunc function():array or object null 返回数据的函数
filter object null 过滤器，作为远程获取数据的参数。
autoBind bool true 是否自动绑定数据，如果为true则插件调用时就会绑定数据，否则需要再插件调用后手动调用bind方法绑定数据。
type string：normal,tree,group normal 数据类型，指定绑定的数据集合是普通列表数据，树形数据，还是分组数据。
theme string null 主题，用于附加样式类到插件上，以定义不同外观。
method string POST 请求远程数据的方式，是get请求还是post请求。
fields object 数据字段配置。
fields.key string 'Id' 主键字段，很多操作都跟主键相关，可配置该字段，对返回数据的格式比较灵活，配置为主键字段的属性取值必须唯一。
fields.total string 'TotalItems' 总记录数字段，当可分页时需要该字段。
fields.items string 'Items' 集合数据字段。
